## Descargar del Modelo de IA (Tier 2)

Por restricciones de almacenamiento, los pesos del modelo Transformer pre-entrenado con fine-tuning no se incluyen en este repositorio. 

Para que la herramienta funcione correctamente, debes descargar el modelo manualmente:
1. Descarga la carpeta comprimida del modelo desde este enlace: https://drive.google.com/drive/folders/1hmUzBmo4WPT3_LLIE7ddHFlxQGVh4uWM?usp=drive_link
2. Descomprime el archivo y colócalo en tu disco local.
3. Abre el archivo `app.py` y modifica la constante `RUTA_MODELO` en la línea 16 para que apunte a la carpeta que acabas de descargar.
